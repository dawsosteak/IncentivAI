**How do I _go hunt_ in Hawaii?**
Pass a basic   
[**Hawaii Hunter Education class**](https://gohunthawaii.ehawaii.gov/public/basic-classes.html) |  |  Pass an out-of-state   
Hunter Education class   
---|---|---  
Receive a Hawaii   
Hunter Certification Card  |  |  Request a Hawaii   
[**Letter of Exemption**](https://hunting.ehawaii.gov/hunting/exemptionRequestForm.html)  
Purchase a   
**[Hawaii Hunting License](https://hunting.ehawaii.gov/hunting/license.html)** |  |  Apply for   
**[Hunts](https://gohunthawaii.ehawaii.gov/public/hunts)**  
**[Purchase Tags & Permits](https://gohunthawaii.ehawaii.gov/public/tags) **  
Check in to a hunting station   
**[Go hunt](http://dlnr.hawaii.gov/recreation/hunting/)!**
