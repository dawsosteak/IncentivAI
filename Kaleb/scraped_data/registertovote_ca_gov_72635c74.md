# ウェブサイトのヘルプ
本ページには、カリフォルニア州オンライン有権者登録 (California Online Voter Registration: COVR) のウェブサイトを最適にご覧いただくための要件に関する情報を記載しています。
## ブラウザ要件
### 対応するブラウザ
COVRのアプリケーションは、Microsoft Internet Explorer (IE) 8.0以上、Opera 10以上、最新バージョンのGoogle Chrome、Mozilla Firefox、またはSafariを使用する場合に最適に動作します。これらのブラウザの旧バージョンをご使用の場合、最新バージョンへの更新が必要かもしれません。ご使用のブラウザの更新は、[https://browsehappy.com/](https://browsehappy.com/)をご覧ください。
### JavaScript
COVRのアプリケーションは、ご使用のウェブブラウザでJavaScriptが有効になっている必要があります。JavaScriptが無効になっていると、オンライン申込書を使用して登録を完了することはできません。[印刷可能な登録用紙をダウンロード](https://www.eac.gov/voter_resources/register_to_vote.aspx)して、それを記入完了の上、郵送できます。
###  PDFリーダー
COVRのウェブサイトには、ご使用のウェブブラウザで直接開くことができないポータブル・ドキュメント・フォーマット (Portable Document Format: PDF) 文書へのリンクが含まれています。無料PDFリーダーのダウンロードとインストールには、次のサイトをご覧ください。 
  * [Adobe社のAcrobat Reader – すべてのプラットフォーム](https://get.adobe.com/reader/)
  * [Nitro PDF Software社のPDFダウンロード - すべてのプラットフォーム](https://www.gonitro.com/pdf-reader)


##  COVRのアプリケーションの使用
### スクリーンリーダーの使用
次のページに進めない場合、必要な質問に返答していないことが考えられます。次に進むには、上からページを読み直し、質問を完了するためのヘルプについての説明メッセージまたは警告メッセージをお聞きください。その質問に返答され次第、インタビューの次のページに進むことができます。
### 特殊文字
COVRのアプリケーションでは、英文字 (例、a、b、c、1、2、3) のみが認められており、次の特殊文字は使用できません。
```
~ $ % ^ & * ( ) + = [ ] { } < > ? / \ |
```

### ブラウザの「戻る」ボタン
ブラウザの「戻る(Back)」ボタンではなく、登録アプリケーションの「前へ (Previous)」および「次へ (Next)」のボタンを使用してください。 
カリフォルニア州州務長官は、ウェブブラウザ、プラグイン、またはその他の推奨されているダウンロードを含め、第三者によるツールのインストールに起因するハードウェアやソフトウェアの障害について一切責任を負いません。
