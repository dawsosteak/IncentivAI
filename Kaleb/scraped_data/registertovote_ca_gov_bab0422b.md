# Accessibility
The Secretary of State is committed to providing a website that is accessible to all visitors. This website was designed to meet the level Double-A of the World Wide Web Consortium (W3C) [Web Content Accessibility Guidelines 2.0](https://www.w3.org/TR/WCAG20/). This website was built using code that is compliant with the W3C guidelines for HTML and CSS. The guidelines help organizations make web content more accessible for people with disabilities.
The Secretary of State's office strives to make all of its web pages as accessible and user-friendly as possible. If you have difficulty accessing the California Secretary of State website, please call the Secretary of State's Voter Hotline at [(800) 345-VOTE (8683)](tel:18003458683) or [email the Secretary of State's Accessibility Coordinator.](https://www.sos.ca.gov/email-accessibility-coordinator/)
## Using a Screen Reader
If you cannot advance to the next page, a required question was missed. To advance, re-read the page from the top to hear any instructional or alert messages for help in completing the question. Once the question is answered, the application will allow you to advance to the next page in the interview.
