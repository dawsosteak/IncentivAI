# Electronic Filing Password Request
Filer ID numbers are now required when requesting passwords for existing campaign and lobbying entities. To look up a campaign or lobbying Filer ID number, or to see if a name already exists, use the search function on [CAL-ACCESS](https://cal-access.sos.ca.gov/). 
Note: Passwords are only needed for state-level filers submitting through Cal-Online. Local-level filers that need an ID number should file a Form 410 following the [instructions provided on the form](https://www.fppc.ca.gov/forms.html).
For new committees with state-level 24-hour filing requirements, please submit a password request form with a completed Form 410 through the [digital filing process](https://www.sos.ca.gov/campaign-lobbying/campaign-filing/digital-signature-filing-instructions-campaign-filings) or by expedited mail.
For assistance with filing electronically, you may contact the Cal-Online Help Desk:
  * Phone: [1-877-745-3453](tel:1-877-745-3453) or [916-653-7283](tel:916-653-7283)
  * Email: prdcalonlinehelp@sos.ca.gov


Important: To use the fillable form and print copies, you must use a PDF reader. For instructions to download a free PDF reader, visit the [Download Free Document Readers](https://www.sos.ca.gov/free-doc-readers "Download Free Document Readers") page.
