R.14-10-003 ALJ/KHY/ 3853778312 PART I: PROCEDURAL ISSUES
* * *
A. Brief description of Decision: This decision, in an effort to move closer to “a consistent universal framework for assessing the cost-effectiveness of all resources, both distributed energy resources and supply side resources” adopts a framework for cost-effectiveness analysis. It adopts the total resource cost (TRC) test as the primary test to be used for distributed energy resource filings or any advice letters that require cost-effectiveness analysis. In addition, any relevant proceedings will also need to include discussion of the program administrator cost (PAC) and ratepayer impact measure (RIM) cost-effectiveness tests. Modified TRC, PAC, and RIM tests will serve as replacements for existing cost tests – replacing the interim greenhouse gas adder values with those adopted in D. 18-02 018. The decision adopts a three-element societal cost test (SCT) consisting of a societal discount rate, an avoided social cost of carbon, and an air quality adder, to be evaluated after an initial data gathering period to end on December 31, 2020. Finally, the decision reaffirms that only minor changes can be made to the avoided cost calculator using the previously-approved resolution process and changes will instead be made via formal process, to commence in August 2019.
