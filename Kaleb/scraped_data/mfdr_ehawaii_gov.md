# Mortgage Foreclosure Dispute Resolution Program
Welcome to the Mortgage Foreclosure Dispute Resolution online filing portal.
To view notices of public sale published pursuant to Hawaii Revised Statutes sections 667-20(2), 667-27(d)(2), or 667-96(d)(2), please click on "Notices of Public Sale", above.
Please select your role from the "Login As" option on the menu bar in order to access your case information.
_[In order to file a foreclosure notice or publish a notice of public sale, an ehawaii.gov subscriber account is required.](http://portal.ehawaii.gov/subscriber-services.html)_
To publish a notice of public sale, select "Publish Notice of Public Sale" from the "Account" menu
This website is optimized for the latest versions of Microsoft Internet Explorer, Mozilla Firefox, or Apple Safari. If you are having problems using the website, please upgrade your browser to the latest version. If the problems persist, please contact us.
