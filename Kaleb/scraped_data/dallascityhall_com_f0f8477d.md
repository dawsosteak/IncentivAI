## Charges for providing Copies of Public Information
1. Standard paper copy (Each side that has recorded Information is considered a page.) @ $.10 per page 2. Nonstandard copy (A) Diskette @ $1.00 each (B) Magnetic Tape @ actual cost (C) Data Cartridge @ actual cost (D) Tape Cartridge @ actual cost (E) Rewritable CD (CD-RW) @ $ 1.00 each (F) Non-rewritable CD (CD-R) @ $ 1.00 each (G) Digital video disc (DVD) @ $ 3.00 each (H) JAZ drive @ actual cost (I) Other electronic media @ actual cost (J) VHS video cassette @ $ 2.50 each (K) Audio cassette @ $ 1.00 each (L) Oversize paper copy (e.g.: 11 inches by 17 inches, Greenbar, bluebar, not including maps and photos using specialty paper) @ $.50 per page (M) Specialty paper (e.g.: Mylar, blueprint, blueline, map, photographic) @ actual cost 3. Personnel charge (A) Programming Personnel @ $35.62 per hour (B) Other Personnel @ $18.75 per hour 4. Overhead charge 20% of personnel 5. Micro Fiche/Film @ actual cost to duplicate or $.10 per page for standard size 6. Remote document retrieval charge @ $ 6.00 per box 7. Computer resource charge (A) Mainframe @ $10.00 per CPU minute (B) Midsize @ $ 1.50 per CPU minute (C) Client/Server @ $ 2.20 per clock hour (D) PC or LAN @ $ 1.00 per clock hour 8. Miscellaneous supplies @ actual cost 9. Postage & shipping charge @ actual cost 10. Photographs @ actual cost
