# 웹사이트 도움말
이 페이지는 캘리포니아 주 온라인 유권자 등록(COVR) 웹사이트의 최적 보기 요구사항에 대한 정보를 제공합니다.
## 브라우저 요구사항
### 브라우저 지원
COVR 애플리케이션은 마이크로소프트 인터넷 익스플로러(Microsoft Internet Explorer, IE) 8.0 이상, 오페라(Opera) 10 이상, 또는 구글 크롬(Google Chrome), 모질라 파이어폭스(Mozilla Firefox), 사파리(Safari)의 최신 버전을 통해서 가장 잘 볼 수 있습니다. 이러한 브라우저들의 이전 버전을 사용하고 있는 경우에는 최신 버전으로 업그레이드해야 할 수도 있습니다. 브라우저를 업그레이드하려면 [https://browsehappy.com/](https://browsehappy.com/)을 찾아가십시오.
### 자바 스크립트(JavaScript)
COVR 애플리케이션은 웹 브라우저에서 자바 스크립트를 활성화할 것을 요구합니다. 자바 스크립트가 활성화되지 않으면 온라인 애플리케이션을 사용하여 유권자 등록을 할 수 없습니다. 이러한 경우에는 [인쇄 가능 등록 양식을 다운로드](https://www.eac.gov/voter_resources/register_to_vote.aspx)하여 작성한 후에 우편으로 보낼 수도 있습니다.
###  PDF 리더(Reader)
COVR 애플리케이션에는 웹 브라우저에서 직접 열 수 없는 이동 가능 문서 형식(Portable Document Format, PDF)으로 작성된 문서로 연결하는 링크가 들어 있습니다. 무료 PDF 리더를 다운로드 및 설치하려면 다음의 웹사이트를 방문하십시오. 
  * [Adobe의 Acrobat Reader - 모든 플랫폼](https://get.adobe.com/reader/)
  * [Nitro PDF 소프트웨어에 의한 PDF 다운로드 - 모든 플랫폼](https://www.gonitro.com/pdf-reader)


##  COVR 애플리케이션의 사용
### 화면 리더의 사용
다음 페이지로 진행할 수 없는 이유는 필수 질문에 답변하지 않았기 때문입니다. 다음 페이지로 진행하려면 이 페이지를 위에서부터 다시 읽고 질문에 답변하는 것을 도와 주는 설명 또는 경보 메시지를 들으십시오. 질문에 답변을 했으면 이 애플리케이션이 이 인터뷰의 다음 페이지로 진행하는 것을 허용합니다.
### 특수 문자
COVR 애플리케이션에서는 영어 문자(a, b, c, 1, 2, 3 등)만을 사용할 수 있고, 다음과 같은 특수 문자들은 사용할 수 없습니다: 
```
~ $ % ^ & * ( ) + = [ ] { } < > ? / \ |
```

### 브라우저의 "뒤로(Back)" 버튼
애플리케이션 내에 있는 "이전(Previous)" 및 "다음(Next)" 버튼을 사용하고, 브라우저의 "뒤로" 버튼을 사용하지 마십시오. 
캘리포니아 주 총무처장관은 다른 회사의 툴(웹 브라우저, 플러그인, 또는 제의를 받아 다운로드한 다른 애플리케이션 포함)을 설치함으로써 발생한 하드웨어나 소프트웨어 문제에 대해 책임을 지지 않습니다.
