# ​Neighborhood Code  

Neighborhood Code Compliance is responsible for addressing nuisance and property standard violations to ensure a safe and healthful environment for the community. Neighborhood Code is divided into seven (7) Community Code Districts with offices throughout the City. For office locations, please see below.

###  **Convenience Store Registration**  

The purpose of this chapter is to protect the health, safety, and welfare of the citizens of the city of Dallas by reducing the occurrence of crime, preventing the escalation of crime, and increasing the successful prosecution of crime that occurs in convenience stores in the city.  

  

For additional information about the Convenience Store registration program, please contact Code Compliance at 214-948-5353 or email: cstore@dallascityhall.com.
Intensive Case Resolution Team   

The Intensive Case Resolution Team (ICR) is designed to assist Neighborhood Code cases where compliance efforts have hit a roadblock, and all enforcement efforts have been exhausted.
