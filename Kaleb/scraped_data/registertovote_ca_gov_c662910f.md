# 網站幫助
本頁提供加州線上選民登記（COVR）網站的最佳瀏覽要求。
## 瀏覽器要求
### 瀏覽器支持
使用Microsoft Internet Explorer (IE) 8.0或更高版本、Opera 10或更高版本，或是Google Chrome、Mozilla Firefox或Safari的最新版本，COVR申請表運行最順暢。如果您使用這些瀏覽器的較舊版本，可能需要升級到最新版本。如欲升級瀏覽器，請瀏覽[https://browsehappy.com/](https://browsehappy.com/)。
### JavaScript
COVR申請表需要您的網路瀏覽器啟動JavaScript。如果JavaScript被禁用，您將無法使用我們的線上申請表完成登記。您可以在[下載可列印的登記表](https://www.eac.gov/voter_resources/register_to_vote.aspx)填寫和郵寄。 
###  PDF閱讀器
COVR申請表包含通往便攜文件格式（PDF）文件的連結，這些文件不能直接在您的網路瀏覽器打開。如欲下載和安裝免費的PDF閱讀器，您可以瀏覽：
  * [Adobe的Acrobat Reader – 所有平台](https://get.adobe.com/reader/)
  * [Nitro PDF Software的PDF Download – 所有平台](https://www.gonitro.com/pdf-reader)


## 使用COVR申請表
### 使用螢幕閱讀器
如果您無法進入下一頁，那是因為您漏填一個必答題。如欲繼續，從頭開始再次閱讀該頁，以聽取幫助回答問題的說明或警示訊息。回答問題後，申請表將允許您進入下一頁的訪談。
### 特殊字元
COVR申請表僅接受英文字元（即a、b、c、1、2、3），不接受以下任何特殊字元：
```
~ $ % ^ & * ( ) + = [ ] { } < > ? / \ |
```

### 瀏覽器「返回」按扭
使用登記申請表中的「上一頁」和「下一頁」按鈕，而非使用瀏覽器「返回」按鈕。
加州州務卿對於安裝第三方工具造成的硬體或軟體問題概不負責，包括網路瀏覽器、插件或其他建議的下載項目。
