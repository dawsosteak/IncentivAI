# Website Help
This page provides information on the California Online Voter Registration (COVR) website’s optimum viewing requirements.
## Browser Requirements
### Browser Support
The COVR application works best when using Microsoft Internet Explorer (IE) 8.0 or higher, Opera 10 or higher, or the latest version of Google Chrome, Mozilla Firefox, or Safari. If you are using an older version of any of these browsers, you may need to upgrade to the latest version. To upgrade your browser, go to https://browsehappy.com/.
### JavaScript
The COVR application requires Javascript to be enabled in your web browser. If JavaScript is disabled, you will not be able to complete your registration using our online application. You may download a printable registration form to complete and mail.
### PDF Reader
The COVR application contains links to documents in the Portable Document Format (PDF), which cannot be directly opened by your web browser. To download and install a free PDF reader, you may visit:
  * [Acrobat Reader by Adobe - all platforms](https://get.adobe.com/reader/)
  * [PDF Download by Nitro PDF Software - all platforms](https://www.gonitro.com/pdf-reader)


## Using the COVR Application
### Using a Screen Reader
If you cannot advance to the next page, a required question was missed. To advance, re-read the page from the top to hear any instructional or alert messages for help in completing the question. Once the question is answered, the application will allow you to advance to the next page in the interview.
### Special Characters
The COVR application only accepts English characters (i.e., a, b, c, 1, 2, 3) and does not accept any of the following special characters:
```
~ $ % ^ & * ( ) + = [ ] { } < > ? / \ |
```

### Browser "Back" Button
Use the "Previous" and "Next" buttons within the registration application and not the browser "Back" buttons.
The California Secretary of State is not responsible for any hardware or software problems resulting from the installation of third party tools, including web browsers, plug-ins, or other suggested downloads.
