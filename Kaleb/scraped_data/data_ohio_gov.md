The Ohio Checkbook is a comprehensive presentation of financial and transactional data maintained in the state's accounting system, which can be found at Checkbook.Ohio.gov.
The DataOhio Portal now provides monthly download files of these dashboards, which are added on a semi-annual basis, to enable further analysis.
To view the latest files, select “Export Data” on the upper right-hand header. For historical annual files, select a zip file below. Once downloaded - you can extract one or more months from the file.
  
Users can filter historical files by “Data Title” or “Date Added”.
