**Here to file your annual business taxes?**   
In March 2018, we launched a new system. You'll need a Secure Access Washington (SAW) user ID and password to log in ([Check if you have one](https://secure.dor.wa.gov/home/Account/GetUserId)). You'll also need to set up access to your tax account in My DOR.
