## The .gov means it's official. 
Local, state, and federal government websites often end in .gov. Commonwealth of Pennsylvania government websites and email systems use "pennsylvania.gov" or "PA.gov" at the end of the address. Before sharing sensitive or personal information, make sure you're on an official state website.

The unified design system for Pennsylvania
# Keystone Design System
Enabling teams across the commonwealth with the tools to quickly build modern, easy-to-use, and accessible digital services.
[ Get started ](https://wcmauthorguide.pa.gov/en/keystone-design-system/getting-started)
![A collage of components used in the Keystone Design System, including icon cards, list groups, menus, a modal, and others.](https://s7d9.scene7.com/is/image/statepa/Figma%20Component%20Library?ts=1743783542685&dpr=off)
## Get to know the Keystone Design System
Check out our guide to getting started, learn about the foundations of the design system, or review documentation for KDS components.
![](https://s7d9.scene7.com/is/image/statepa/patterns)
### Get started with KDS
The Keystone Design System provides principles, guidance, and code to help you design and build accessible, mobile-friendly websites and digital services for the Commonwealth of Pennsylvania.
[ Get started with KDS ](https://wcmauthorguide.pa.gov/en/keystone-design-system/getting-started)
![](https://s7d9.scene7.com/is/image/statepa/components)
### Components
Learn when and how to use the components that make up the Keystone Design System.
[ Components ](https://wcmauthorguide.pa.gov/en/keystone-design-system/components)
![](https://s7d9.scene7.com/is/image/statepa/KDS-Foundations)
### Foundations
Foundations are the visual elements that make up engaging layouts and end-to-end user experiences.
[ Foundations ](https://wcmauthorguide.pa.gov/en/keystone-design-system/foundations)
