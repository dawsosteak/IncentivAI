# **Instructions for using the draft CCR report:**
## **New CCR reports for the**
**1.** Select your PWS ID# and CCR year from the drop down lists, then click the "View Report" button. The report will generate and the first page will be visible. The report cannot be edited from this website, it must be downloaded. To download your report, select the blue floppy disc icon. The best file type choices are PDF or Word.
**2.** Review the yellow highlighted or red italicized text and add/verify.
**3.** Review all the data for accuracy and completeness. If errors are discovered, please [contact us](https://deq.mt.gov/files/Water/PublicWaterSupplyContacts_Oct2022.pdf).
**4.** If your system used purchased water, ensure all applicable water quality data and violations from the wholesale system(s) are included with the report.
**5.** If you were issued a monitoring waiver and wish to provide the required public notice in the CCR, you can either: 
**5a.** Include the public notice form provided with your waiver approval letter as a separate page in the CCR, or 
**5b.** Add text that includes the contaminant the waiver was granted for, the period for which the monitoring waiver is in effect, and the reason for the waiver.
**6.** If applicable, a blank UCMR or PFAS data table will appear. Fill out the data table using the provided instructions.
**6a.** Radon- report the results of monitoring and an explanation of the significance of the results in a separate data table (DEQ can help with language if needed). 
**7.** Complete the blank table for Turbidity (if applicable, a table will be in the report).
**8.** Complete the blank table for RTCR Assessments (if applicable, a table will be in the report).
**9.** If using the report to meet Tier 3 public notice requirements:
**9a.** Ensure the 10 required elements are met by using the completed public notice template that was included with your violation letter as a separate page in the CCR.
**10.** Add information about any uncorrected significant deficiencies, and the plan or schedule for correction, and any interim measures taken.
**11.** Deliver your report to your consumers using approved delivery methods by June 30, 
**12.** Submit to DEQ: 
**12a. A copy of the report** you delivered to your consumers no later than June 30, 
**12b. The CCR certification form** no later than September 30, [here](https://deq.mt.gov/files/Water/Forms/CCR_Certification_Form_2026.docx).
