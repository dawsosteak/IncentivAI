# Current Processing Dates
Processing dates vary depending on the type of request submitted, how the request is submitted, and when the request is received by the California Secretary of State's office.
As significantly more requests are submitted at the end of the fiscal and calendar years, the processing time for all requests takes longer. If a faster response time is needed for a filing request, see [Service Options](https://www.sos.ca.gov/business-programs/business-entities/service-options/ "service options") for information about expediting a filing request.
