#  Administrative Letters for California Teleconnect Fund
Below are the California Teleconnect Fund program's Administrative Letters to participating carriers. Please refer to the [CTF Service Provider Manual](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/guidebooks/ctf_service_provider_manual---updated---january-26.pdf) for the current applicable information from these letters.
  * [Administrative Letter No. 32](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/california-teleconnect-fund-administrative-letter-no-32.pdf) - CTF eCAP Update- School and Library Annexes
  * [Administrative Letter No. 31](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter--31-ecap.pdf) - Requires CTF service providers collect participant contact information for eCAP
  * [Administrative Letter No. 30](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no30.pdf) - Requires CTF service providers to submit claims via eCAP
  * [Administrative Letter No. 29](https://files.cpuc.ca.gov/California_Teleconnect_fund/Website_Reference/Administrative_Letter_No._29.pdf "Administrative Letter No. 29") - Introduction of the updated CTF Claim Workbook
  * [Administrative Letter No. 28(a)](https://files.cpuc.ca.gov/California_Teleconnect_fund/Website_Reference/Administrative%20Letter_No._28\(a\).pdf "Administrative Letter No. 28\(a\)") - Updated Eligible Service List and Data Request CTF-001
  * [Administrative Letter No. 27](https://files.cpuc.ca.gov/California_Teleconnect_fund/Website_Reference/Administrative_Letter_No._27.pdf "Administrative Letter No. 27") - Modified Application Numbers for Private Schools
  * [Administrative Letter No. 26](https://www.cpuc.ca.gov/-/media/cpuc-website/files/uploadedfiles/cpuc_public_website/content/utilities_and_industries/communications-telecommunications_and_broadband/consumer_programs/california_teleconnect_fund/admin-letter-26.pdf "Administrative Letter No.26") - CTF Eligible Services List
  * [Administrative Letter No. 25](https://www.cpuc.ca.gov/-/media/cpuc-website/files/uploadedfiles/cpuc_public_website/content/utilities_and_industries/communications-telecommunications_and_broadband/consumer_programs/california_teleconnect_fund/administrative-letter-no-25.pdf "Administrative Letter No.25") - Changes to CTF voice discounts pursuant to D.16-04-021
  * [Administrative Letter No. 24](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no024-2015.pdf) - Schools receiving discounts from the Federal E-rate Program
  * [Administrative Letter No. 23](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no023-2015.pdf) - Clarifying Appendix C of D.15-07-007 
  * [Administrative Letter No. 22](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no022-2011.pdf) - Revised California Teleconnect Fund Application
  * [Administrative Letter No. 21](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no021-2011.pdf) - 30-day Limit to Respond to Deficiency Letters
  * [Administrative Letter No. 16](https://www.cpuc.ca.gov/-/media/cpuc-website/files/uploadedfiles/cpuc_public_website/content/utilities_and_industries/communications-telecommunications_and_broadband/consumer_programs/california_teleconnect_fund/administrativeletterno16-1-.pdf "Administrative Letter No. 16") - Implementation of CTF Changes Pursuant to D.08-06-020
  * [Administrative Letter No. 14](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no014-2006.pdf) - BEAR Funding Method
  * [Administrative Letter No. 13](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no013-2006.pdf) - Carriers must File Advice Letters to Reflect CTF Tariff Provisions
  * [Administrative Letter No. 10b](https://www.cpuc.ca.gov/-/media/cpuc-website/files/uploadedfiles/cpuc_public_website/content/utilities_and_industries/communications-telecommunications_and_broadband/consumer_programs/california_teleconnect_fund/al10bjanuary222008.doc "Administrative Letter No. 10b") - Application E-rate Discount (SB 1102)
  * [Administrative Letter No. 9](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/california-teleconnect-fund/administrative-letters/administrative-letter-no009-2005.pdf) - Revised method for calculating discounts for less than a one-month period
