#  California Advanced Services Fund (CASF)
**[Postponement of the Deadline for Action on Pending 2025 CASF Infrastructure Applications and 2026 CASF Infrastructure Project Application Deadline](https://www.cpuc.ca.gov/-/media/cpuc-website/divisions/communications-division/documents/casf-infrastructure-and-market-analysis/casf-infra-letter---postponement-of-2025--deem-deny-deadline-and-2026-app-cycle.pdf)**
**March 23, 2026 - **By this letter, the California Public Utilities Commission (CPUC) extends the deadline for approvals of pending 2025 CASF Infrastructure applications from May 15, 2026 to July 31, 2026 and provides updates regarding the 2026 CASF Infrastructure application deadline.
**Announcing the new Electronic Claims and Applications Portal (eCAP):** Effective September 26, 2023, the CASF program is now using the [Electronic Claims and Applications Portal (eCAP)](https://ecap.cpuc.ca.gov/s/), an electronic system for CASF grantees to submit payment requests to the CPUC, replacing paper based-submissions. If grantees have any questions or need assistance, contact the respective CASF Account contacts.
### Program Description and Application Instructions
The California Advanced Services Fund (CASF) is allocated to the following CASF accounts:
  * [Broadband Adoption Account ](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/casf-adoption-account)
  * [Broadband Infrastructure Grant Account](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/casf-infrastructure-grant)
  * [Broadband Public Housing Account](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/casf-public-housing-account)
  * [Rural and Urban Regional Broadband Consortia Grant Account ](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/casf-consortia-account)
  * [Line Extension Pilot Program](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/casf-line-extension-program)
  * [Tribal Technical Assistance Program](https://www.cpuc.ca.gov/industries-and-topics/internet-and-phone/california-advanced-services-fund/tribal-technical-assistance)
