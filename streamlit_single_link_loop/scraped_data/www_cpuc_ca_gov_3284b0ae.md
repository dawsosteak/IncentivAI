#  Emergency Load Reduction Program (ELRP) Data and Information
This page serves for Emergency Load Reduction Program (ELRP) data and information. For details about program parameters and how to participate in ELRP, please go to the [main ELRP webpage here](https://www.cpuc.ca.gov/industries-and-topics/electrical-energy/electric-costs/demand-response-dr/emergency-load-reduction-program).
* * *
**In 2021, the California Public Utilities Commission (CPUC) created an innovative program, the Emergency Load Reduction Program (ELRP), to pilot a new Demand Response approach to help avoid rotating outages during peak summer electricity usage periods from May thru October. The ELRP started in 2021 with commercial customer participation. In December 2021, the CPUC expanded the program to include residential customers for Summer 2022 and beyond.**
### ELRP Performance and Dispatch Data
ELRP has been dispatched multiple times in summers of 2021, 2022, and 2023, lowering electricity demand statewide during critical conditions when the state needs it most. Data concerning these dispatches can be found below.

### ELRP Procedural History (R.20-11-003, A.22-02-005 et al.)
  * [R.20-11-003](https://apps.cpuc.ca.gov/apex/f?p=401:56:0::NO:RP,57,RIR:P5_PROCEEDING_SELECT:R2011003)
    * [D.21-03-056](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M373/K745/373745051.PDF): Established ELRP pilot program for summers 2021-2025
      * [Attachment 1](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M373/K973/373973362.PDF): ELRP pilot program guidance and parameters
    * [D.21-06-027](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M389/K525/389525818.PDF): Clarifying Group A day-of and day-ahead triggers
      * [Attachment 1](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M389/K525/389525728.pdf): Redline modifications to D.21-03-056
    * [D.21-12-015](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M428/K821/428821475.PDF): ELRP Phase II adding residential customers and VGI aggregators to Group A, adding day-of trigger to group B, increasing compensation rate to $2/kWh, and more.
      * [Attachment 2](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M428/K821/428821668.PDF): ELRP pilot program Phase II guidance and parameters
    * [D.21-12-069](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M434/K616/434616630.PDF): Executive Director's order correcting inadvertent errors in D.21-12-015
  * [A.22-02-005, et al.](https://apps.cpuc.ca.gov/apex/f?p=401:56:0::NO:RP,57,RIR:P5_PROCEEDING_SELECT:A2205002)
    * [ALJ ](https://docs.cpuc.ca.gov/PublishedDocs/Efile/G000/M502/K977/502977248.PDF)[Jungreis Ruling](https://docs.cpuc.ca.gov/PublishedDocs/Efile/G000/M502/K977/502977248.PDF): Provided ELRP program data for 2022 season (found in the Ruling in pdf form)
    * [D.23-12-005](https://docs.cpuc.ca.gov/PublishedDocs/Published/G000/M521/K486/521486520.PDF): Extended nonresidential ELRP through 2027, lowered compensation rate for power saver rewards to $1/kWh of incremental load reduction, reduced the event window for A.4 (VPP) and A.5 (VGI) from 5 hours to 3 hours, et al.
