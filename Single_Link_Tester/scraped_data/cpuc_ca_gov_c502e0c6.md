# News Release Search Form
Commission News Releases, since July 2000, are available online. **Click[ here](https://docs.cpuc.ca.gov/SearchRes.aspx?NewsSearch=true&DaySearch=30) to access the last 30 days of releases or use the search function below to search the archives.**

Publication date: From |  To: (use mm/dd/yyyy)  
* * *
## SEARCH TIPS:
You may use one or all three search fields to search News Releases.


  * **Document title** : enter all or part of a news release title. For example entering - Edison - will pull up all news releases that include Edison in the title  

  

  * **Industry:** select an industry from the drop down list.  

  

  * **Publication Date:** enter the date range, for example - from 09/01/01 to 10/31/01.
